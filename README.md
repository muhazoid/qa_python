

## **Тестирование приложения BooksCollector**

Данный проект содержит набор автоматизированных тестов для проверки функциональности класса BooksCollector. Класс предназначен для управления коллекцией книг, включая добавление книг, установку жанров, формирование списка избранного и фильтрацию книг для детей.

## **Структура проекта**

main.py — содержит основной класс BooksCollector с логикой приложения.
conftest.py — файл с фикстурами (подготовка данных и экземпляров класса для тестов).
tests.py — файл с тестами.

## **Описание класса BooksCollector**

Класс реализует следующие возможности:

***Добавление книг:*** 
Книги добавляются без жанра. Название должно быть от 1 до 40 символов. Дубликаты игнорируются.

***Установка жанра:*** 
Жанр можно установить только из предопределенного списка (Фантастика, Ужасы, Детективы, Мультфильмы, Комедии).

***Фильтрация для детей:*** 
Есть метод, возвращающий книги, жанры которых подходят для детей (исключаются Ужасы и Детективы).

***Избранное:*** 
Книги можно добавлять в избранное и удалять оттуда.

## **Описание тестов (tests.py)**

В классе TestBooksCollector реализованы следующие тестовые методы:

***test_add_new_book_add_two_books***
Проверяет базовую функциональность добавления книг.
Действие: Добавляет две разные книги.
Ожидаемый результат: В словаре books_genre содержится ровно 2 записи.

***test_add_new_book_valid_names***
Проверяет валидацию названий книг при добавлении (параметризованный тест).
Входные данные: Названия разной длины (40 символов), с пробелами, дефисами, цифрами.
Ожидаемый результат: Книги успешно добавляются в словарь books_genre.

***test_add_new_book_not_add_duplicate***
Проверяет защиту от добавления дубликатов.
Действие: Дважды добавляется книга с одинаковым названием.
Ожидаемый результат: В словаре остается только одна запись.

***test_set_book_genre_valid_genres_good_add***
Проверяет установку допустимых жанров (параметризованный тест).
Действие: Книге назначается жанр из списка разрешенных.
Ожидаемый результат: Жанр корректно сохраняется и возвращается методом get_book_genre.

***test_set_book_genre_invalid_genre_not_set***
Проверяет защиту от установки несуществующего жанра.
Действие: Попытка назначить книге жанр, которого нет в списке разрешенных.
Ожидаемый результат: Жанр не устанавливается (значение остается пустой строкой).

***test_get_books_with_specific_genre_good_get***
Проверяет фильтрацию книг по конкретному жанру.
Действие: Добавляются книги с разными жанрами (используется рандомизация для части книг).
Ожидаемый результат: Метод get_books_with_specific_genre возвращает список, содержащий только книги жанра "Фантастика".

***test_get_books_with_specific_genre_invalid_genre_not_add***
Проверяет поведение метода при запросе несуществующего жанра.
Действие: Запрос книг с жанром "Invalid genre".
Ожидаемый результат: Возвращается пустой список.

***test_get_books_for_children_good_get***
Проверяет получение списка книг, подходящих для детей.
Предусловие: Используется фикстура collector_with_books, где часть книг имеет "взрослые" жанры (Ужасы, Детективы).
Ожидаемый результат: Возвращается список только с книгами жанров Фантастика, Мультфильмы, Комедии. Книги с жанрами Ужасы и Детективы исключены.

***test_add_book_in_favorites_adds_book***
Проверяет добавление книги в избранное.
Действие: Книга добавляется, затем добавляется в избранное.
Ожидаемый результат: Метод get_list_of_favorites_books возвращает список с этой книгой.

***test_delete_book_from_favorites_removes_book_good_remove***
Проверяет удаление книги из избранного.
Действие: Книга добавляется в избранное, затем удаляется оттуда.
Ожидаемый результат: Список избранного пуст.



## Используемые фикстуры (conftest.py)

**
Для изоляции и упрощения тестов используются следующие фикстуры:

***collector:*** 
Создает "чистый" экземпляр класса BooksCollector перед каждым тестом.

***collector_with_books:*** 
Создает экземпляр класса и предзаполняет его 5 книгами с разными жанрами для проверки фильтров.

Запустить тесты из терминала можно такой командой:
pytest -v tests.py
